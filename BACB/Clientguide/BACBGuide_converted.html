2.2 Candidate communications

What communications does the client send to candidates?
BACB sends an Authorization to Test (ATT) email when candidates are approved to schedule an exam.
What communications does Pearson VUE send to candidates?
- Pearson VUE sends a confirmation letter whenever candidates schedule, reschedule, or cancel an exam.
- Pearson VUE sends an order payment receipt whenever candidates purchase an exam.
- Pearson VUE sends an email when the candidate's web-based score report is available (excluding paper-and-pencil exams).

2.3 Accommodations and accessibility

Who approves accommodations?
BACB
What's the request process?
- Candidates must submit a request form on the BACB website: https://www.bacb.com/accommodations-documentation/
- If candidates need help with the request process, direct them to contact BACB.

2.4 Scheduling

What's the scheduling process?
Candidates must follow these steps:
1. Submit a certification application to BACB: https://www.bacb.com/
2. Receive an Authorization to Test (ATT) email from BACB.
3. Schedule and pay for the exam through either method:
- Pearson VUE website
- Customer Service (Pearson VUE Call Center)

2.5 Rescheduling

What's the rescheduling process?
Candidates can reschedule through either method:
- Pearson VUE website
- Customer Service (Pearson VUE Call Center)

How late can candidates reschedule test center appointments?
48 hours

How many times can candidates reschedule the same appointment?
Unlimited up to 48 hours before the appointment

Is there a fee to reschedule?
It depends on the timing. See the table below:

Time prior to appointment Exam length Fee (USD)
More than 30 days Any length No fee
5 to 30 days 1.5 hours $27.00
4 hours $59.00
2 to 5 days 1.5 hours $32.00
4 hours $69.00

Who should candidates contact to reschedule due to an emergency?
- If the appointment is more than 48 hours away, candidates should follow the standard reschedule process.
- If the appointment is less than 48 hours away, direct candidates to contact BACB: https://www.bacb.com/appeals/administrative-appeals/

5.4 Required ID

How many IDs are required?
One primary and one secondary ID

What are the primary ID requirements?
The primary ID must be government-issued and unexpired. The ID must also have a photograph and a signature. Only the following IDs are acceptable:

  • Driver's license
    Note: Temporary driver's licenses are permitted if they have all required elements.
  • Passport
  • Military ID
    Note: Military IDs without a visible signature are acceptable at military test centers only.
  • Alien registration card (green card, permanent resident card, or visa)
  • State/country ID card

What are the secondary ID requirements?
The secondary ID must be unexpired and have a signature. Only the following IDs are acceptable:

  • Anything on the primary ID list
  • Library card
  • Social Security card
  • Bank card
  • Employee ID

6.5 Candidate misconduct

How should TAs respond to candidate misconduct?
If you suspect or witness candidate misconduct, follow these steps:

  1. If possible, have a coworker witness the candidate's behavior.
  2. Intervene to deal with the situation:
    • If the candidate is being admitted, do not allow the candidate to begin testing.
    • If the candidate is in the testing room, put the exam in unscheduled break mode and escort the candidate from the testing room.
    • If the candidate is taking an unscheduled break, do not let the candidate back into the testing room.
  3. Inform the candidate of the following:
    • They cannot continue testing because they have violated a policy. (Explain which policy they violated, but do not use the words "cheating" or "misconduct.")
    • They need to leave the test center.
    • They should contact BACB for additional information.
    • They may reschedule the exam at their own expense if BACB allows it.
  4. If the candidate’s exam has already started, put the exam into unscheduled break mode if you haven't already. Allow the exam to continue running until it times out.
  5. Call the test center support line to report the situation to a program coordinator.
  6. Create a case, selecting "Exam delivery" and "Candidate error or misconduct."

6.6 Technical issues

How should TAs handle technical issues?
Follow the standard process. Please note the scenarios below:

Unscheduled Break Issue (Blue Screen)

Occasionally, a blue screen might appear when you put an exam in unscheduled break mode or when a candidate returns from an unscheduled break. If this occurs, you cannot resume the exam or log back in to Delivery Manager.

  1. Before you call VSS, reboot the delivery workstation. When you reboot the workstation, the exam clock stops so that candidates do not lose more time.
  2. Log in to Delivery Manager and try to resume the exam. If the exam resumes, skip to step 4.
  3. If the exam will not resume after rebooting the workstation, call VSS. VSS will help troubleshoot the issue, but they will not add more time to the exam because the exam clock stops when you reboot the workstation.
  4. Create a case, selecting "Exam delivery" and "Problem with Pearson VUE application."

Internet Connection Issues

All BACB exams require an internet connection, so internet outages and slowdowns may affect the exams. Occasionally, an exam may take a minute or two to load when first launched or when a candidate returns from an unscheduled break. Please give each exam a few minutes to load before contacting VSS or restarting the exam. Pressing Ctrl+R on the keyboard to refresh the browser page will resolve most issues triggered by network latency or spikes in server traffic.

Always create a case if a candidate experiences connection issues, even if the issues are resolved without contacting VSS or restarting the exam. When you create the case, select "Exam Delivery" and "Problem with Pearson VUE application."
